## KDE Plasma dots...

Ensure you have a fresh (or maybe not so fresh) installation of KDE Plasma. And have the following dependencies installed:
- **Theme**: Breeze Default
- **Colors**: [KDE Material You Colors](https://github.com/luisbocanegra/kde-material-you-colors)
- **Window Decorations**: [Klassy](https://github.com/paulmcauley/klassy)
- **Icons**: [Tela Circle Icon Theme Standard](https://github.com/vinceliuice/Tela-circle-icon-theme)
- **Window Rounding and Borders**: [KDE Rounded Corners](https://github.com/matinlotfali/KDE-Rounded-Corners)
- **[Wallpaper](https://unsplash.com/photos/a-small-bird-perched-on-a-thin-branch-OLv0kaeekGo)**

For Arch Linux (assuming you have paru as AUR helper installed, or you can use any of your preferred AUR helper):
```
paru -S kde-material-you-colors klassy-bin tela-circle-icon-theme-standard kwin-effect-rounded-corners kwin-effects-better-blur-dx
```

In your home directory, clone this repository into it:

`cd` into the cloned directory and perform `stow kde`.

!!! **IMPORTANT** !!!

Please manage the conflicts manually, until the automated solution is provided.
